# Plan 1.1: Project Skeleton Scaffolding

## Objective
Establish the foundational frontend and backend architecture, setting up React (Vite/TS/Tailwind) and Node.js (Express/TS/MongoDB).

## Context
- .gsd/SPEC.md
- .gsd/ARCHITECTURE.md

## Tasks

<task type="auto">
  <name>Initialize Backend Project</name>
  <files>
    backend/package.json
    backend/tsconfig.json
    backend/server.ts
    backend/.env
  </files>
  <action>
    - Initialize backend directory with `npm init -y`.
    - Install Express, TypeScript, Mongoose, dotenv, cors, bcryptjs, jsonwebtoken, zod and type definitions.
    - Set up `tsconfig.json` for Node 18+ (outDir: dist).
    - Create a basic `server.ts` file that imports express, connects to MongoDB (stubbed), and starts the server.
    - Export a `backend/.env.example` file with stubbed MONGO_URI, JWT_SECRET, PORT.
  </action>
  <verify>cd backend && npx tsc --noEmit</verify>
  <done>Backend project relies purely on TypeScript and Express server runs locally.</done>
</task>

<task type="auto">
  <name>Initialize Frontend Project</name>
  <files>
    frontend/package.json
    frontend/vite.config.ts
    frontend/tailwind.config.ts
    frontend/src/main.tsx
    frontend/src/index.css
  </files>
  <action>
    - Use Vite to scaffold a React TS project inside `frontend`.
    - Install TailwindCSS and configure `tailwind.config.ts`, `postcss.config.js`, and `src/index.css`.
    - Install `react-router-dom`, `lucide-react` (for icons), and `axios`.
    - Clean up default Vite boilerplate (App.tsx, App.css).
  </action>
  <verify>cd frontend && npx tsc --noEmit</verify>
  <done>Frontend project builds via Vite without TS errors and has Tailwind configured.</done>
</task>

## Success Criteria
- [ ] Backend is securely scaffolded with standard Typescript REST setup.
- [ ] Frontend can build via Vite, and Tailwind globals are active.
